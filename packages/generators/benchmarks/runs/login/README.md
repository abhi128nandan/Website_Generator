## Features
- # Login Form SRS

## Requirements
- Render a form with `email` and `password` inputs.
- Include a submit button.
- Accept an `onSubmit` prop that receives the email and password values.
- Must use React state to manage input values.
- Must be a functional component exported as default.


## Architecture
This is a **frontend-only** React/Vite application.
No backend server or database is required.

## Prerequisites
- Node.js >= 18
- pnpm >= 9

## Getting Started

```bash
# 1. Install dependencies
pnpm install

# 2. Start development server
pnpm run dev
```
