# BasicCalculatorApp
Type: frontend-app
Mode: Frontend Application (No Backend/Database)

## Features
- Arithmetic operations (+, -, *, /)
- Decimal number calculations
- Error handling for division by zero
- Continuous calculation support
- Keyboard input compatibility
- Responsive UI for mobile/desktop

## Architecture
This is a **frontend-only** React/Vite application.
No backend server or database is required.

## Prerequisites
- Node.js >= 18
- pnpm >= 9

## Getting Started

```bash
# 1. Install dependencies
pnpm install

# 2. Start development server
pnpm run dev
```

## Services
| Service  | Port | Command |
|----------|------|---------|
| Frontend | 5173 | `pnpm --dir frontend dev` |
