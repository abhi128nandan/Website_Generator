# BasicCalcApp
Type: frontend-app
Mode: Frontend Application (No Backend/Database)

## Features
- Perform addition, subtraction, multiplication, and division
- Clean responsive UI with calculator display at top
- Number buttons 0-9 and operator buttons (+, -, *, /)
- Equals button to show calculation result instantly
- Clear button to reset input

## Architecture
This is a **frontend-only** React/Vite application.
No backend server or database is required.

## Prerequisites
- Node.js >= 18
- pnpm >= 9

## Getting Started

```bash
# 1. Install dependencies
pnpm install

# 2. Start development server
pnpm run dev
```
