## Features
- # Counter Component SRS

## Requirements
- Render a component with a numeric counter.
- Must have an "Increment" button.
- Must have a "Decrement" button.
- Must use `useState` to store the counter value.
- Must be a functional component exported as default.


## Architecture
This is a **frontend-only** React/Vite application.
No backend server or database is required.

## Prerequisites
- Node.js >= 18
- pnpm >= 9

## Getting Started

```bash
# 1. Install dependencies
pnpm install

# 2. Start development server
pnpm run dev
```
