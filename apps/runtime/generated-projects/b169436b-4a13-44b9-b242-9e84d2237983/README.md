# BasicCalcApp
Type: frontend-app
Mode: Frontend Application (No Backend/Database)

## Features
- Perform addition, subtraction, multiplication, and division
- Responsive calculator UI with display, number buttons (0-9), operator buttons (+, -, *, /), equals, and clear button
- Instant calculation result display on equals press

## Architecture
This is a **frontend-only** React/Vite application.
No backend server or database is required.

## Prerequisites
- Node.js >= 18
- pnpm >= 9

## Getting Started

```bash
# 1. Install dependencies
pnpm install

# 2. Start development server
pnpm run dev
```
